# Pinnacle Frontend

Frontend applications for the Pinnacle platform.

## Structure

This folder contains multiple frontend applications:

- **web/** - Web application (Next.js with TypeScript)
- **app/** - Mobile application (React Native/Flutter)

## Development Workflow

Each application (web/app) will have its own:

- `package.json` - Dependencies and scripts
- Development server configuration
- Build configuration
- Testing setup

## API Integration

All frontend applications will communicate with the backend API at:

- Development: `http://localhost:3000/api`

## Contributing

Follow the conventional commits format for all commits.
Ensure code passes linting and formatting checks before pushing.
